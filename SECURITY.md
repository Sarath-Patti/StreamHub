# Security Policy

## Supported Versions

The following versions of StreamHub are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

We take the security of StreamHub seriously. If you discover a security vulnerability, please do NOT open a public GitHub issue.

Instead, please report security concerns directly to the maintainers:

1. **Email**: Send a detailed security report to `sarathpatti@users.noreply.github.com`
2. **Details**: Include a summary of the vulnerability, potential impact, and step-by-step proof-of-concept steps.
3. **Response**: We will acknowledge receipt within 48 hours and work on a security fix.

Thank you for helping keep StreamHub and our user community safe!
