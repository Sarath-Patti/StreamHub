# Contributing to StreamHub

Thank you for considering contributing to StreamHub!

## Code Style & Standards

- Maintain strict TypeScript type safety.
- Format code using Prettier and verify linting with ESLint before opening pull requests.
- Keep module interfaces clean and decoupled.

## Workflow

1. Fork and clone the repository.
2. Create a feature branch.
3. Ensure linting passes cleanly.
4. Submit a descriptive Pull Request.
